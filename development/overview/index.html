Overview

Aside from the firmware, the application consists of 3 parts:

  • flowcontrol
  • flowcontrol-monitor
  • flowcontrol-cli

flowcontrol and flowcontrol-cli are currently independent, I’m looking to consolidate their codebase now they’re both using Go and utilise shared packages for stuff like serial communication.

flowcontrol

This is the GUI application. It’s built using wails

flowcontrol-monitor

This is the supplementary application to the main one. It provides system metrics such as GPU Clock and temperatures using LibreHardwareMonitorLib. I’m aiming to make this an optional component for people who want hardware metrics beyond the scope of the controller, it’s currently bundled with the release.

flowcontrol-cli

A CLI application built using cobra
